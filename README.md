# VentiAI

A safe space for emotional expression powered by AI.

## About

VentiAI is a web-based application where you can come to vent your frustrations, grief, and emotions in a supportive environment. Built with React, Node.js, and Bootstrap, the app processes your thoughts and provides empathetic emotional support tailored to help you feel heard and understood.

**Important Disclaimer**: VentiAI is designed to provide emotional support and make you feel good and supported. However, it is **not a substitute for therapy or medical advice**. If you are experiencing a mental health crisis, please contact a qualified healthcare professional or call a crisis helpline.

## Installation

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Setup

1. Clone the repository:
```bash
git clone https://github.com/RanjanKhadka012/VentiAI.git
cd VentiAI
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

## Usage

1. Open the application in your web browser
2. Share your thoughts, feelings, or frustrations in the text area
3. Submit your message
4. Receive supportive, empathetic responses from the AI
5. Continue the conversation as needed to feel heard and supported
