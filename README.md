# VentiAI

VentiAI is a web-based application built using Java, CSS, Python with React, Node.js, and TensorFlow.js. It provides a safe space where users can vent their frustrations and emotions, receiving supportive AI-generated feedback to help them feel better. The app also features speech-to-text capabilities, allowing users to simply speak their frustrations and have them converted into text form for processing.

## Features

- **Emotional Venting Platform**: Express your frustrations and emotions in a safe, judgment-free environment
- **AI-Powered Support**: Receive personalized, supportive feedback from AI to help improve your emotional well-being
- **Speech-to-Text**: Speak your thoughts and have them automatically converted to text
- **Multi-Technology Stack**: Built with modern technologies for a robust and responsive user experience

## Technology Stack

- **Frontend**: React, CSS
- **Backend**: Node.js, Java, Python
- **AI/ML**: TensorFlow.js
